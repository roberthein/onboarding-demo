# Onboarding Demo

A small iOS onboarding experience built with a focus on polished motion, adaptable layout, and clean visual styling.

## What I focused on

- **Dynamic layouts**: views are designed to adapt smoothly across device sizes and orientations. Tested on iPhones and iPads in landscape and portrait.
- **Smooth scrolling**: paging and transitions are tuned to feel responsive and fluid.
- **Themeable UI**: styling is structured to be easy to swap and iterate.

## Tech notes

- Built with Swift and UIKit components. (And a little bit of SwiftUI)
- Uses reusable onboarding pages and shared theme/style primitives.
